Diamond Ribs


Row 1: edge st, * p1, k4, (p1, k1) x 6, p1, k4; repeat from * to last 2 sts, p1, edge st
Row 2 & all other wrong side rows: edge st, purl across to last st, edge st
Row 3: edge st, * p2, k3, p1, k1, p1, k2, (p1, k1) x 2, p2, k1, p1, k3, p1; repeat from * to last 2 sts, p1, edge st
Row 5: edge st, * k1, p2, k2, p1, k1, p1, k3, (p1, k1) x 3, p1, k2, p2; repeat from * to last 2 sts, k1, edge st
Row 7: edge st, * k2, p2, k1, p1, k1, p2, k3, p1, k1, p2, k1, p1, k1, p2, k1; repeat from * to last 2 sts, k1, edge st
Row 9: edge st, * k1, p2, k2, (p1, k1) x 2, p1, k3, (p1, k1) x 2, p1, k2, p2; repeat from * to last 2 sts, k1, edge st
Row 11: edge st, * p2, k3, p1, k1, p2, k1, p1, k3, p2, k1, p1, k3, p1; repeat from * to last 2 sts, p1, edge st
Row 13: edge st, * p1, k4, (p1, k1) x 6, p1, k4; repeat from * to last 2 sts, p1, edge st
Row 15: edge st, * p2, k3, p1, k1, p2, (k1, p1) x 2, k2, p1, k1, p1, k3, p1; repeat from * to last 2 sts, p1, edge st
Row 17: edge st, * k1, p2, k2, (p1, k1) x 3, p1, k3, p1, k1, p1, k2, p2; repeat from * to last 2 sts, k1, edge st
Row 19: edge st, * k2, p2, k1, p1, k1, p2, k1, p1, k3, p2, k1, p1, k1, p2, k1; repeat from * to last 2 sts, k1, edge st
Row 21: edge st, * k1, p2, k2, (p1, k1) x 2, p1, k3, (p1, k1) x 2, p1, k2, p2; repeat from * to last 2 sts, k1, edge st
Row 23: edge st, * p2, k3, p1, k1, p2, k3, p1, k1, p2, k1, p1, k3, p1; repeat from * to last 2 sts, p1, edge st
